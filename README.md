```bash
computation (n.)

    c. 1400, "act, process, or method of arithmetical calculation," from Latin
    computationem (nominative computatio), noun of action from past participle
    stem of computare "to sum up, reckon, compute," from com "with, together"
    (see com-) + putare "to reckon," originally "to prune," from PIE root *pau-
    (2) "to cut, strike, stamp." From 1713 as "a result of computation, amount
    reckoned."
```
